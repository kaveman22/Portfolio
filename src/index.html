Facial Recognition Application

This is a full stack application built on React JS, Node JS, and PostGreSQL. The Core Features here are a sign in and register page, and a homepage that allows the user to enter a picture file (.jpg url). It utilizes the Clarifai API to conduct a facial recognition algorithm and then draws a blew box around a face in the picture that was submitted. It stores a count of the number of pictures that a user has submitted as well as user login information. All passwords are encrypted ans stored as hashes in the database.
